Shell - loops, conditions and parsing

1-for_best_school: displays "Best School" 10 times using a for loop
2-while_best_school: displays "Best School" 10 times using a while loop
3-until_best_school: displays "Best School" 10 times using an until loop
4-if_9_say_hi: displays "Best School" 10 times, with "Hi" after the 9th
5-4_bad_luck_8_is_your_chance: loops 1-10, prints bad/good luck on 4th/8th
6-superstitious_numbers: loops 1-20, prints superstition messages via case
7-clock: displays the time for 12 hours and 59 minutes
8-for_ls: lists directory contents, showing names after the first dash
9-to_file_or_not_to_file: checks existence, emptiness, and type of "school"
10-fizzbuzz: displays numbers 1-100 with FizzBuzz substitutions
11-read_and_cut: displays username, uid, and home dir from /etc/passwd
12-tell_the_story_of_passwd: tells a story for each user in /etc/passwd
13-lets_parse_apache_logs: extracts visitor IP and HTTP status from Apache logs
14-dig_the-data: groups visitors by IP and HTTP code, sorted by occurrence
